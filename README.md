# Caesar Cipher

A Python program that encrypts and decrypts text using the Caesar Cipher algorithm.

## Features

Encrypt messages
Decrypt messages
User-friendly menu
Supports uppercase and lowercase letters

## How to Run

1. Open terminal
2. Navigate to the project folder
3. Run:

python caesar_cipher.py

## Example

Message: hello
Shift: 3

Encrypted: khoor

Encrypted Message: khoorcat 
Shift: 3

Decrypted: hello
